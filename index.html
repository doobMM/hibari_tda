hibari-TDA 인터랙티브 데모

사카모토 류이치의 hibari를 Topological Data Analysis (Persistent Homology)로 분석하고, 발견된 cycle 구조를 시드 삼아 음악을 생성하는 연구 파이프라인의 브라우저 데모입니다. 연구 전체 내용은 GitHub 저장소와 환대_포트폴리오.pdf를 참조하세요.

Visualization

Tonnetz 시각화

hibari 원곡을 재생하면서 Tonnetz 격자 위에 H1 cycle overlay를 동기 표시합니다. 파이프라인 7단계 MIDI(v0 원곡 + v1~v6 생성본)가 페이지에 인라인되어 있어 다운로드 없이 버튼 한 번으로 재생됩니다. Playback · Controls · Appearance · Sound · TDA 5개 탭에서 색상·음향·사이클 강조 방식을 조정할 수 있습니다.

Overlap Matrix 편집 + 음악 생성

DFT α=0.25 per-cycle τ 기준 참조 overlap matrix (T=1088 × K=14)를 브라우저에서 직접 편집하고, 편집본을 시드 삼아 30초 × 3개 후보 음악을 즉석 생성해 재생·MIDI 저장할 수 있습니다. Algorithm 1 (샘플링, JS=0.00902) · Algorithm 2 (FC ONNX) 둘 다 지원.
