What is this demo? 이 데모는?

This demo visualizes hibari by Ryuichi Sakamoto on the Tonnetz, a diagram representing tonal space used in music theory and TDA research.

The semi-transparent polygons are H₁ cycles (1-dimensional topological features) detected by Persistent Homology applied to the note co-occurrence structure of hibari. They represent groups of notes that form persistent loops in the Vietoris–Rips complex.

Based on cifkao/tonnetz-viz by Ondřej Cífka.

이 데모는 류이치 사카모토의 hibari를 Tonnetz 위에 시각화합니다. Tonnetz는 음악 이론과 TDA 연구에서 사용하는 음 공간 다이어그램입니다.

반투명 다각형들은 H₁ 사이클입니다 — hibari 음표의 공기화(co-occurrence) 구조에 지속적 호몰로지(Persistent Homology)를 적용해 검출한 1차원 위상수학적 특징으로, Vietoris–Rips 복합체에서 지속되는 루프를 나타냅니다.

cifkao/tonnetz-viz (Ondřej Cífka) 기반.

How to use 사용법
  • Open the Playback tab and press Play hibari — no file needed.
  • Notes light up on the Tonnetz as the piece plays.
  • H₁ cycle polygons are always visible as colored overlays.
  • Use a hardware MIDI device (Controls tab) or computer keyboard for live input.
  • Scroll to zoom; Transform overlay (bottom-left) and Interval overlay (right) are always visible.
  • 재생 탭을 열고 Play hibari를 클릭하세요 — 파일 선택 불필요.

  • H₁ 사이클 다각형은 항상 색깔 오버레이로 표시됩니다.
  • 하드웨어 MIDI 장치(연주 탭) 또는 컴퓨터 키보드로도 연주할 수 있습니다.
  • 스크롤로 줌; 변환 오버레이(좌하단)와 인터벌 오버레이(우측)는 항상 표시됩니다.
Neo-Riemannian Transforms P · L · R · N · S · H 네오리만 변환 P · L · R · N · S · H

Neo-Riemannian theory describes parsimonious voice-leading between triads — each operation moves exactly one note by one or two semitones while keeping the other two fixed. On the Tonnetz, each operation corresponds to a reflection of the current triangle across one of its edges.

OpWhat movesFormulaExample (C major → ?)Tonnetz
PThe third (±1 semitone){r,r+4,r+7} ↔ {r,r+3,r+7}C maj {C,E,G} → C min {C,E♭,G}Flip across top edge
LThe root (−1 semitone){r,r+4,r+7} → {r+4,r+7,r+11}C maj {C,E,G} → E min {E,G,B}Flip across right edge
RThe fifth (+2 semitones){r,r+4,r+7} → {r,r+4,r+9}C maj {C,E,G} → A min {A,C,E}Flip across left edge

Each operation is its own inverse: P(P(x))=x, L(L(x))=x, R(R(x))=x. P preserves root, L preserves the minor third, R preserves the major third.

Composite operations (N, S, H)
OpCompositionExample (C major → ?)Character
NR → L → PC maj → F minNebenverwandt — neighboring triad a P4 below
SL → P → RC maj → D♭ majSlide — root moves by one semitone, quality unchanged
HL → P → LC maj → A♭ majHexatonic pole — tritone-related triad, same quality

In the demo: press any button while notes are lit (or during playback). The current chord is identified as the best-matching major/minor triad, the operation is applied, and the result flashes for 2.5 s. The label (e.g. L → E min) fades in the overlay.

네오리만 이론은 삼화음 사이의 절약적 성부진행을 기술합니다 — 각 연산은 정확히 한 음을 1~2 반음 이동하고 나머지 두 음은 고정합니다. Tonnetz 위에서 각 연산은 현재 삼각형을 한 변(edge)을 기준으로 반사(reflection)시키는 것에 해당합니다.

각 연산은 자기 역원입니다: P(P(x))=x, L(L(x))=x, R(R(x))=x. P는 근음을, L은 단3도를, R은 장3도를 보존합니다.

Transpositions Tn 이조 (Transposition) Tn

A transposition Tn shifts every pitch class by n semitones modulo 12 — the simplest and most familiar musical transformation.

Tn(p) = (p + n) mod 12

On the Tonnetz, Tn slides the entire highlighted region along one of the grid's axes:

  • ±1 — chromatic semitone (m2); diagonal move
  • ±7 — perfect fifth (P5); horizontal move in Riemannian layout
  • ±12 — octave; pitch classes unchanged visually (audio shifts one octave)

Continuous effect: unlike the one-shot PLR operations, the transposition offset persists. Use the 0 (reset) button to return to the original key.

Example: hibari is in G major. T+5 transposes it to C major. The lit pattern shifts five steps left on the Tonnetz.

이조 Tn은 모든 음높이 클래스를 n 반음만큼 이동시킵니다 (mod 12) — 가장 기본적이고 친숙한 음악적 변환입니다.

Tn(p) = (p + n) mod 12

Tonnetz 위에서 Tn은 점등된 전체 영역을 격자의 한 축 방향으로 이동시킵니다:

  • ±1 — 반음 (m2); 대각선 방향 이동
  • ±7 — 완전5도 (P5); 리만 레이아웃에서 수평 이동
  • ±12 — 옥타브; 음높이 클래스 시각적으로 동일 (오디오만 한 옥타브 이동)

연속 효과: PLR의 일회성 연산과 달리, 이조 오프셋은 지속됩니다. 0 (초기화) 버튼으로 원래 조성으로 돌아옵니다.

예시: hibari는 G장조입니다. T+5를 적용하면 C장조로 이조됩니다. Tonnetz 위 점등 패턴이 왼쪽으로 다섯 칸 이동합니다.

Inversions In 반전 (Inversion) In

An inversion In reflects all pitch classes around an axis n — the combination of transposition and a mirror flip of the pitch-class circle.

In(p) = (2n − p) mod 12

  • In is its own inverse: applying it twice returns to the original.
  • It maps a major triad to a minor triad — the harmonic shape is preserved but mirrored.
  • The axis n acts as a fixed point: if p = n then In(p) = p.
AxisMaps C major {C,E,G} to…Musical interpretation
I0 (C){C, A♭, F} = F minInvert around C
I4 (E){E, C, A} = A min= R (Relative)
I7 (G){G, E♭, C} = C min= P (Parallel)
I11 (B){B, G, E} = E min= L (Leading-tone)

Notice: I4 = R, I7 = P, I11 = L when applied to major triads. The Neo-Riemannian operations are special cases of pitch-class inversion. On the Tonnetz, inversion corresponds to a global reflection of the grid rather than a local edge-flip.

In the demo: toggle ON and click an axis note. All subsequent MIDI events are remapped by In. Combine with transpose to reach any inversion in the standard TI-group.

반전 In은 모든 음높이 클래스를 축 n을 중심으로 반사시킵니다 — 이조와 음높이 클래스 원의 거울 뒤집기를 결합한 변환입니다.

In(p) = (2n − p) mod 12

  • In은 자기 역원입니다: 두 번 적용하면 원래로 돌아옵니다.
  • 장화음을 단화음으로 대응시킵니다 — 화성적 형태는 보존되지만 거울처럼 뒤집힙니다.
  • 축 n은 고정점 역할을 합니다: p = n이면 In(p) = p.
축C장조 {C,E,G}의 대응음악적 해석
I0 (C){C, A♭, F} = F단조C 축 반전
I4 (E){E, C, A} = A단조= R (관계조 전환)
I7 (G){G, E♭, C} = C단조= P (동명조 전환)
I11 (B){B, G, E} = E단조= L (이끔음 교환)

핵심: I4 = R, I7 = P, I11 = L (장화음에 적용 시). 네오리만 연산 P/L/R은 사실 음높이 클래스 반전의 특수한 경우입니다. Tonnetz 위에서 반전은 국소적 변 뒤집기가 아닌 격자 전체의 반사에 해당합니다.

데모에서: ON 토글 후 축 음이름을 클릭합니다. 이후 모든 MIDI 이벤트에 In이 적용됩니다. 이조 오프셋과 조합하면 표준 TI-군의 모든 반전에 도달할 수 있습니다.
